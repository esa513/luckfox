# AFE4404

## 調整 TIA 增益

* 查看可用增益（預設500000） 

    ```
    cat /sys/bus/iio/devices/iio:device1/in_intensity_resistance_available
    ```

* 增益調成 10000

    ```
    for i in 1 2 3 ; do echo 10000 > /sys/bus/iio/devices/iio:device1/in_intensity${i}_resistance ; done
    ```

* 開光並讀值

    ```
    for i in 1 2 3 ; do echo 0 > /sys/bus/iio/devices/iio:device1/out_current${i}_raw ; done ; echo 10 > /sys/bus/iio/devices/iio:device1/out_current1_raw ; cat /sys/bus/iio/devices/iio:device1/in_intensity{1..3}_raw
    for i in 1 2 3 ; do echo 0 > /sys/bus/iio/devices/iio:device1/out_current${i}_raw ; done ; echo 10 > /sys/bus/iio/devices/iio:device1/out_current2_raw ; cat /sys/bus/iio/devices/iio:device1/in_intensity{1..3}_raw
    for i in 1 2 3 ; do echo 0 > /sys/bus/iio/devices/iio:device1/out_current${i}_raw ; done ; echo 10 > /sys/bus/iio/devices/iio:device1/out_current3_raw ; cat /sys/bus/iio/devices/iio:device1/in_intensity{1..3}_raw
    ```
